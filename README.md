# Sales Analytics Dashboard (Power BI)

## 📊 Overview
This project is an interactive Sales Analytics Dashboard built using Power BI to analyze sales performance, product trends, and regional insights.

## 🚀 Features
- KPIs:
  - Year-over-Year (YoY) Growth %
  - Profit Margin %
  - Average Order Value
- Product-level insights (Top products by orders and profit)
- Category and subcategory analysis
- Regional performance analysis (map visualization)
- Interactive filtering and dynamic visuals

## 🛠️ Tools & Technologies
- Power BI
- DAX (Data Analysis Expressions)
- Data Modeling
